# Malvasia Nera

"Malvasia Nera" è un sinonimo utilizzato per diversi tipi di [Malvasia](/vitigni/bacca-bianca/malvasia). Non sempre le informazioni fornite dai produtti permettono di risalire al vitigno originale.
